weather-web (Search only in Indian State and Popular Place)
This is a simple Weather App that I built using HTML, CSS, and JavaScript.
With this app, I can search the weather of any city or even a small place in India.
All the weather data comes from the OpenWeather API.

Features

Search weather of any place in India
Shows current temperature
Weather icon
Weather description
Min and Max temperature
Feels Like temperature
Humidity
Wind speed
Visibility
Sunrise & Sunset timing
Real-time date and time
Technologies Used

HTML CSS JavaScript REST openweather API

How to Use

Type the name of the city in the search bar
Click the search button
The weather details will appear instantly
If you type a wrong city name, it shows “City not found”

API Used REST Countries API https://openweathermap.org/api
